# Assets Folder

## Logo Instructions

Untuk mengganti logo dashboard:

1. Simpan file PNG logo Anda dengan nama `logo.png` di folder ini
2. Pastikan file PNG memiliki:
   - Resolusi yang baik (minimal 48x48 px, disarankan 96x96 px atau lebih tinggi)
   - Background transparan untuk hasil terbaik
   - Format PNG untuk kualitas terbaik

3. Setelah menempatkan file `logo.png`, refresh browser untuk melihat perubahan

## Format yang didukung:
- PNG (direkomendasikan untuk logo dengan background transparan)
- JPG/JPEG (untuk logo dengan background solid)
- SVG (akan bekerja, tapi perlu penyesuaian CSS)

## Ukuran yang disarankan:
- Minimal: 48x48 px
- Optimal: 96x96 px atau 128x128 px
- Maksimal: 256x256 px (untuk kualitas tinggi)

Logo akan otomatis di-scale agar fit dalam area 48x48 px pada header.
